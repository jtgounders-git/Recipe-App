# 🍲 Recipe Manager – WPF Desktop Application
A desktop recipe management application built with **C# and Windows Presentation Foundation (WPF)** that allows users to create, organize, edit, and manage recipes with ingredients, calorie tracking, and filtering.

---

# 📖 Overview

Recipe Manager enables users to store and manage a collection of recipes while tracking ingredient quantities, food groups, and calorie values.

Each recipe includes:

- Ingredients with quantities and measurement units  
- Food group classifications  
- Calorie calculations  
- Preparation instructions  

The application supports dynamic scaling of ingredients and provides filtering tools to help users locate recipes based on dietary needs.

---

# ✨ Features

## 📝 Recipe Creation & Management

Users can create unlimited recipes with:

- Unique recipe names  
- Detailed preparation instructions  
- Ingredient lists with quantities and measurement units  
- Calorie values for each ingredient  
- Food group categorization  

## 🧮 Ingredient Scaling

Ingredients can be scaled dynamically, automatically adjusting:

- Quantities  
- Total calorie calculations  

## 🔎 Recipe Search & Filtering

Recipes can be filtered by:

- Ingredient search  
- Food group  
- Maximum calorie limit  

## 📋 Recipe List Management

Recipes are stored in a structured list and displayed alphabetically for easy browsing.

## ✏️ Recipe Editing

Users can update recipes at any time, and changes are immediately reflected.

## ⚠️ Calorie Notifications

Alerts when total calorie value exceeds a threshold using C# delegates for event-driven notifications.
