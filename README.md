\# 🍲 Recipe Manager – WPF Desktop Application



!\[C#](https://img.shields.io/badge/Language-C%23-blue)

!\[Framework](https://img.shields.io/badge/Framework-WPF-purple)

!\[Platform](https://img.shields.io/badge/Platform-.NET-green)



A desktop recipe management application built with \*\*C# and Windows Presentation Foundation (WPF)\*\* that allows users to create, organize, edit, and manage recipes with detailed ingredients, calorie tracking, and intelligent filtering.



The system demonstrates strong \*\*object-oriented design, data management, and user interface development using XAML\*\*.



---



\# 📖 Overview



Recipe Manager enables users to store and manage a collection of recipes while tracking ingredient quantities, food groups, and calorie values.



Each recipe includes:



• Ingredients with quantities and measurement units  

• Food group classifications  

• Calorie calculations  

• Preparation instructions  



The application supports dynamic scaling of ingredients and provides filtering tools to help users quickly locate recipes based on dietary needs.



---



\# ✨ Features



\## 📝 Recipe Creation \& Management



Users can create unlimited recipes with:



• Unique recipe names  

• Detailed preparation instructions  

• Ingredient lists with quantities and measurement units  

• Calorie values for each ingredient  

• Food group categorization  



---



\## 🧮 Ingredient Scaling



The system allows ingredients to be scaled dynamically.



Scaling automatically adjusts:



• Ingredient quantities  

• Total calorie calculations  



This makes it easy to adapt recipes for different serving sizes.



---



\## 🔎 Recipe Search \& Filtering



Recipes can be filtered using multiple criteria:



• Ingredient search  

• Food group filtering  

• Maximum calorie limit  



These filters allow users to quickly locate recipes that match dietary requirements.



---



\## 📋 Recipe List Management



All recipes are stored in a structured list and displayed in \*\*alphabetical order\*\*, making it easy for users to browse and manage their recipe collection.



---



\## ✏️ Recipe Editing



Users can update recipes at any time by modifying:



• Ingredient quantities  

• Ingredient details  

• Preparation instructions  



Changes are immediately reflected in the application.



---



\## ⚠️ Calorie Notifications



The system alerts users when the total calorie value of a recipe exceeds a defined threshold.



This feature is implemented using \*\*C# delegates for event-driven notifications\*\*.



---



\# 🧠 Software Design Principles



The project demonstrates several core software engineering concepts:



• Object-Oriented Programming  

• SOLID design principles  

• DRY (Don't Repeat Yourself) coding practices  

• Modular class design for recipes and ingredients  

• Delegates and events for notification systems  



Unit testing is also included to validate calorie calculations and ensure application reliability.
